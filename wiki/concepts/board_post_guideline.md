# 헤이스트(HASTE) 게시판 글쓰기 규격서 (Board Posting Guidelines)

이 문서는 헤이스트 점주/공지 게시판의 일관된 스타일링과 동적 기능 작동을 위해 정의된 공식 글쓰기 규격입니다. 앞으로 모든 게시글 작성 시 이 형식을 준수하여 작성해야 합니다.

---

## 📌 1. 소제목 작성 규격 (Bold 자동 변환)
본문의 가독성을 높이기 위해 문단별 소제목은 반드시 **대괄호 `[...]`** 형식으로 작성합니다.
*   **규격**: `[소제목 내용]`
*   **주의사항**: 대괄호 앞뒤로 줄바꿈(`\n`)을 두어 독립된 한 줄로 작성하십시오.
*   **예시**:
    ```text
    [멤버십 단계별 가입 절차]
    1단계: 매장 코드 인증 및 정보 입력...
    ```
*   **렌더링 효과**: 게시판 뷰어에서 자동으로 굵은 글씨(`!font-bold !text-stone-900`)로 강조되며 상하 여백이 배치됩니다.

---

## 📌 2. 인터랙티브 팝업 버튼 삽입 규격 (예약어)
본문 중간에 모달 팝업창을 여는 버튼을 생성하고 싶을 경우, 본문 텍스트 내에 정확하게 아래의 예약어(Trigger Text)를 그대로 삽입합니다.

| 팝업 종류 | 본문 삽입 예약어 (공백 및 괄호 일치 필수) | 연동되는 동작 |
| :--- | :--- | :--- |
| **가이드북 팝업** | `[가이드북 팝업 열기]` | 클릭 시 4단계 가입 가이드북 팝업 호출 |
| **혜택안내 팝업** | `[혜택안내 팝업 열기]` | 클릭 시 정기 구독 플랜별 혜택안내 팝업 호출 |

*   **예시**:
    ```text
    단계별 상세 설명서 및 가이드는 아래 링크를 클릭하여 확인하세요.
    [가이드북 팝업 열기]
    ```

---

## 📌 3. 본문 단락 정렬 및 스타일 규칙
1.  **가독성 여백**: 각 단락 및 소제목 구역 사이에는 1줄의 빈 줄(Double Newline)을 추가하여 모바일에서도 쾌적하게 보이도록 조정합니다.
2.  **리스트 구조화**: 상세 세부 사항은 하이픈(`-`) 또는 숫자(1, 2, 3)를 활용하여 정렬합니다.
